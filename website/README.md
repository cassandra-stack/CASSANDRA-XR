# CASSANDRA XR Docusaurus

This folder contains the documentation site scaffold for CASSANDRA XR.

## Usage

Install dependencies from `website/`:

```bash
npm install
```

Start the local site:

```bash
npm run start
```

Build the static site:

```bash
npm run build
```

The site reads documentation pages from the repository-level `docs/` folder so the Markdown source of truth stays shared with the codebase.
